# Music Playlist App

A React web application that allows users to search for music, create custom playlists, and manage their music library.

## Live Demo

[View Live Demo](https://stellar-rugelach-3d47ba.netlify.app)

## Purpose

This project demonstrates core React skills: component architecture, state management, API integration, and deployment. It showcases the ability to build interactive applications, handle user input, manage complex state flows, and integrate third-party APIs.

## Features

- Search for songs by title using MusicBrainz API
- Create and customize playlists
- Add and remove tracks from playlists
- Rename playlists
- Responsive design for desktop and mobile

## Technologies

- React
- MusicBrainz API
- Netlify
- Git

## Installation

1. Clone the repository from GitHub
2. Navigate to the project directory
3. Run `npm install` to install dependencies
4. Run `npm start` to start the development server

## Usage

1. Enter a song title in the search bar
2. Click "Search" to find matching tracks
3. Click "+ Add" to add songs to your playlist
4. Click on the playlist title to rename it
5. Click "- Remove" to remove songs from your playlist

## Notes

Originally built with Spotify API. Switched to MusicBrainz due to free tier limitations.
